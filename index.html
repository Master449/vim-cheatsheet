Text Manipulation

i insert before cursor r replace single character
I insert at start of line cc replace line
a insert after cursor cw replace to end of word
A insert at end of line c$ replace to end of line
o add new line below cursor s substitute character
O add new line above cursor S substitute line
ea insert at end of word u undo
Esc exit insert mode Ctrl + r redo
H jump to first line in document
j move cursor up

Visual Mode

v enter visual mode ~ change case
V enter linewise visual mode Esc exit visual mode
CTRL + v start visual block mode
> shift text left
< shift text right
>> shift left by shiftwidth
<< shift right by shiftwidth
== auto-indent line
0 B b h
move to start of line move to start of previous word (puncuation) move to start of previous word move cursor left
l w W $
move cursor right move to start of next word move to start of next work (puncuation) move to end of line
y yank/copy D delete to end of line
yy yank a line x delete character
yw yank a word /string search for "string"
y$ yank to end of line
p paste after cursor
P past before cursor
dd delete/cut a line
dw delete a word

Text Manipulation Cont.

j move cursor down
G jump to last line in document
ZZ save and quit
ZQ quit without saving
:w write / save
:q quit
:wq write and quit
:x write and quit
:q! force quit without saving
:qa quit all vim buffers
